# Simple Exress handlebars starter pack

This is a custom express + handlebars starter pack I use for simple applications

## Pages

- Home
- Login
- Register
- Contact
- Profile
- Forgot password
- About
- Contact
- Admin Interface

## Features

- Fully functional web server + web app
- Handlebars templating engine
- UI with Bootstrap
- User login
- User signup
- Email confirmation
- Error handling
- Momentjs
- Authentization
- Authorization
- ES6 && ESNext
- Input Validation
- Tests
- Reset Password / Forgetten Password
